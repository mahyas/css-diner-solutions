What is CSS Diner:
CSS Diner is a fun game to practice CSS selectors by answering 32 questions. It’s very user friendly, and I highly suggest you to try it!
Link to the project:

You can see the project here: https://flukeout.github.io/#
And you can find the repository here: https://github.com/flukeout/css-diner
